Project Description

The Samples project provides relevant sample applications to demonstrate the Java EE Technolgy using the Java EE SDK built by project GlassFish. These sample applications also ship as part of Java EE SDK. Contributing to this project means contributing to the SDK and being part of a bundle which has millions of downloads. The sample applications also work with Sun Glassfish Enterprise Server. These sample applications conform to the BluePrints project guidelines and may be used as templates to start a project or learn a specific Java EE technology.

Learn the JavaEE 6 technologies

  • These samples illustrate some of the JavaEE 6 technologies.
  • Click on the sample from the list to learn about the sample and the procedure to build and run it.
  • Make sure you have downloaded and installed Java EE 6 SDK or GlassFish v3.
  • JavaEE 6 SDK already includes samples, so there is no need to install the samples separately.
  • For GlassFish v3, please follow the instructions given below. In these instructions, <javaee.home> refers to the path where you installed GlassFish v3.
    • Download the samples installer, called javaee6-samples-1.0-b03-installer.jar in subsequent instructions.
    • In a terminal windows, go to the <javaee.home>/samples directory:
      cd <javaee.home>/samples
      (Create the samples directory if it does not exist.)
    • Install the samples by executing the following command:
      java -jar javaee6-samples-1.0-b03-installer.jar
      Study and accept the license agreement to install the samples.
  • Refer to samples/javaee6/index.html for instructions on using sample applications. A simple, one-time setup—common for all samples—is needed to build and run the sample applications. Click on a sample in the list to learn about that sample and the procedure to build and run it.
  • Report issues to users@glassfish-samples.dev.java.net.

Developing sample applications

  • Understand the governance policy for information on roles and responsibilities.
  • All contributors must sign and return a Contributor Agreement.
  • Make sure you have downloaded and installed Java EE SDK or GlassFish.
  • Refer to Java EE 6 Sample Applications Development Process for checking out samples workspace and developing sample application.
  • Report issues to dev@glassfish-samples.dev.java.net.
