Samples for the GlassFish project

The Samples project is meant to demonstrate the Java EE Technolgy using Java EE SDK and GlassFish by providing relevant sample applications. These sample applications conform to the BluePrints project guidelines and may be used as templates to start a project or learn a specific Java EE technology.

If you are interested in using sample applications:
  • Make sure you have downloaded and installed Java EE SDK or GlassFish.
  • Download the Java EE Samples bundle and expand the zip file.
  • Refer to glassfish-samples/ws/javaee5/index.html for instructions on using sample applications.
If you are interested in developing sample applications:
  • Make sure you have downloaded and installed Java EE SDK or GlassFish.
  • Refer to Sample Applications Development Process for checking out samples workspace and developing sample application.
