Sample Applications

Sample applications include the following Java EE 6 samples. Install them to <install-dir>/samples.

Setting Up Your Environment

Make sure you have downloaded and installed Java EE SDK or GlassFish.

Before working with the sample applications, you must set up Application Server environment. Follow the basic steps in the Common Build Instructions for Sample Applications  to configure your environment.

Using Databases with the Samples

Some sample applications require a database. You can use any JDBC-compliant database, but setup scripts and instructions are provided for Java DB. The Java DB database is installed with Sun GlassFish Enterprise Server. It includes a pre-populated database containing tables and data for the sample applications. For more information, see Using Java DB with the Samples.

Getting Help
