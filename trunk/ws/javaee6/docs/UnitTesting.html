Unit Testing for Sample Applications

Each sample application needs one or more test cases so that the sample applications go through regular testing to notice any regressions and for the purposes of promoting the samples bundle. Refer to the test directory of representative sample hello-stateless-ejb for an example.

Here are the steps:

  1. Create a test directory for the sample.
  2. Download and install junit4.
  3. Update the junit.home property in bp-project/test.properties with the junit4 install location.

    junit.home=/home/junit4.0

  4. Create build.xml under test directory whose contents will be similar to the following.

    <import file="../../../../bp-project/test.xml"/>
    <target name="runtest" depends="compile">
            <echo message="Running the sample application test..." />
            <java classname="org.junit.runner.JUnitCore" fork="on">
                    <arg value="enterprise.hello_stateless_ejb_test.HelloStatelessEjbTest"/>
                    <classpath refid="classpath"/>
            </java>
    </target>

  5. Create the java code needed for testing under src/java.
  6. Perform the test by running ant target runtest.
