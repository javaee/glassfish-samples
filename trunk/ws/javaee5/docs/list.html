Sample Application Description
Enterprise Java Beans 3.0
servlet-stateless-ear

This is a very simple EJB 3.0 Stateless Session Bean with a Local business interface that is invoked from a Servlet. This sample demonstrates the concept of dependency injection - one of the new features of Java EE 5.0.

interceptor-stateless-ear

This is a very simple EJB 3.0 Stateless Session Bean with a Remote business interface that uses EJB 3.0 interceptor mechanism to validate the parameter passed to a business method.
lottery-annotation-ear

This is a very simple example intended to demonstrate Annotation facility introduced in Java EE 5.
