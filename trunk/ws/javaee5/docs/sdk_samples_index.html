Sun Java System Application Server 

Sun Java System Application Server Samples

The following sample applications are installed by default:
  • Quickstart Sample - Start here to get up and running quickly on the Application Server. The Quick Start Guide explains how to perform basic tasks such as starting the server, accessing the Admin Console, and deploying a sample application.
  • Java EE 5 Samples - Demonstrate the features of Java Enterprise Edition 5.  Includes detailed instructions to build and run the sample applications.

You can also download J2EE 1.4 Samples that demonstrate features of the J2EE 1.4 platform.  The download includes detailed instructions to build and run the sample applications. Expand the zip file in the directory <appserver-install-root>/samples to create the directory <appserver-install-root>/samples/j2ee14.
