Sample Applications

This document provides important information about the sample applications. Please review it before using the samples.

Samples, Blueprints, and Tutorials

By default the samples include Quickstart,  and javaee5. The samples for j2ee14 is available as a separate download which can be downloaded and installed under <appserver-install-root>/samples.

Refer to the Samples Index and by clicking on the sample you can see its documentation including an overview and the build instructions.

Blueprints

The Java BluePrints are a now part of the java.net community. Refer to blueprints.dev.java.net for more information. 

Tutorials

The Java EE Tutorial is available for download and browsing online. 

Setting Up Your Environment

Make sure you have downloaded and installed Java EE SDK or GlassFish.

Before working with the sample applications, you must set up Application Server environment. Follow the basic steps in the Common Build Instructions for Sample Applications  to configure your environment.

Using Databases with the Samples

Some sample applications require a database. You can use any JDBC-compliant database, but setup scripts and instructions are provided only for the following:

  • Java DB: The Java DB is installed with the Application Server. It includes a pre populated database containing tables and data for the sample applications. For more information, see Using Java DB with the Samples.

Getting Help

If you encounter problems while running the samples, view the Sample Applications topic in the Application Server area of Sun Software Forums - Developer Forums.
