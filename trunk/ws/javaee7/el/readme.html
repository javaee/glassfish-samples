Expression Language 3.0 Sample

A sample demonstrating some new features in Expression Language 3.0.

Description

One of the new features in EL 3.0 is the support for standalone environment. This sample runs without a Javaee container.

Key Features

  • Support for standalone environment
  • Static field or method reference
  • New operators: cat, semicolon, and assignment
  • Lambda expression
  • Collection constructions and operations

Building, Deploying, and Running the Application

The sample executes standalone, without a web container.

Perform the following step to clean, build and run the application:

  1. mvn clean compile exec:java
