Troubleshooting

Include any pertinent troubeshooting info here. Below is a template: add/modify as appropriate.

If you have problems when running the application, try the following:

  • Review the log files, as described in Building, Deploying, and Running Sample Applications.
  • Make sure the database is running. To reset your database, run the command asant sql if using Derby
  • To redeploy the application, you must undeploy it first. Use the commands asant undeploy, asant clean and then asant all.
