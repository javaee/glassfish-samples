Sample Applications

Sample applications include Java EE 5 samples.

Additional J2EE 1.4 samples are available as a separate download. Install them to <install-dir>/samples.

Setting Up Your Environment

Make sure you have downloaded and installed Java EE SDK or GlassFish.

Before working with the sample applications, you must set up Application Server environment. Follow the basic steps in the Common Build Instructions for Sample Applications  to configure your environment.

Using Databases with the Samples

Some sample applications require a database. You can use any JDBC-compliant database, but setup scripts and instructions are provided for Java DB. The Java DB database is installed with the Application Server. It includes a pre-populated database containing tables and data for the sample applications. For more information, see Using Java DB with the Samples.

Getting Help

If you encounter problems while running the samples, view the Sample Applications topic in the Application Server area of Sun Software Forums - Developer Forums.
