Sun Java System Application Server 

Sun Java System Application Server Samples


  • Quickstart Sample
    Useful primarily for confirming successful installation and operation of Application Server.

    Before working with the samples, follow the Quick Start Guide to familiarize yourself with the features and basic operation of Application Server.

  • Java EE 5 Samples
    Contains samples demoing the features of Java EE 5.

  • J2EE 1.4 Samples
    Contains the samples corresponding to previous version of J2EE 1.4.

    By default the j2ee14-samples are not installed.

    It can be downloaded from Sun Developer Network (SDN)

    Expand the downloaded zip file under <appserver-install-root>/samples so that it looks like <appserver-install-root>/samples/j2ee14-samples.

Each of the above links point to the list of samples and detailed instructions to build and run those samples.
